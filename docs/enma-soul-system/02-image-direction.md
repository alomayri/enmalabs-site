# 02. Image Direction

Goal:
- create a hero image that opens the room without looking AI-generated

Best current generation route:
- first pass: Gemini MCP
- refinement/edit pass: fal.ai, especially a context/edit-capable model

Why:
- Gemini is already proven in this repo's hero workflow
- fal is stronger as an edit bench and model-lab after one composition is close

Image brief:

- one subject only
- right-of-center composition
- strong left-side negative space for typography
- warm candlelight
- believable material surfaces
- painterly still life with photographic credibility
- no visible UI
- no fantasy symbols
- no text in image
- no neon glow
- no cool palette
- no decorative clutter

Material vocabulary:
- dark wood
- glass
- paper
- wax
- linen
- brass
- dust
- smoke residue

Avoid:
- generic AI hyperdetail
- multiple unrelated objects competing for meaning
- luxury spa photography
- startup desk shots
- purple/blue gradients
- over-obvious occult imagery

Generation workflow:

1. Generate 3 disciplined compositions with Gemini MCP.
2. Reject anything that feels “pretty” before it feels true.
3. Pick one keeper for composition only.
4. Use fal.ai only if a close image needs structural edits rather than a full restart.
5. Never promote an image because it looks expensive. Promote it because it feels inevitable.
