# 05. Interaction, Motion, Accessibility

Interaction should feel careful, not busy.

Motion is allowed when it does one of these:
- opens the room
- clarifies state
- guides attention
- supports memory
- softens a transition

Motion is not allowed when it exists only to prove taste.

Rules:

- one major moving idea per section
- reduced motion must be real, not symbolic
- long-running atmospheric motion should have a clear fallback
- form feedback must be explicit and local to the field
- keyboard focus must remain visible and predictable
- links and buttons must tell the user where they go

Immediate fixes already implied by this rule set:

- make important labels visible
- avoid placeholder-only fields
- make journal links uniquely named
- verify fixed nav tab order on desktop and mobile
- keep the closing scene alive without turning it into a spectacle

Enma interaction standard:
- soft
- legible
- calm
- responsive
- never coy about what happens next
