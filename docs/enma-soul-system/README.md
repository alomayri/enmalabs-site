# Enma Soul System

This folder is the upstream working sequence for Enma Labs.

It exists so creative direction, storytelling, proof, motion, image generation,
and system rules are defined in one place before more homepage changes get made.

Read in this order:

1. `00-thesis.md`
2. `01-proof-before-poetry.md`
3. `02-image-direction.md`
4. `03-narrative-sequence.md`
5. `04-visual-system.md`
6. `05-interaction-motion-accessibility.md`
7. `06-execution-roadmap.md`

Use this folder with:
- `docs/CREATIVE-DIRECTION.md`
- `AGENTS.md`
- `src/lib/content.ts`
- `src/lib/design-system.ts`

Rule of use:
- If a new idea improves atmosphere but weakens product truth, it does not ship.
- If a new idea improves clarity but flattens the emotional room, refine it until both hold.
- Balsam is the first proof surface. Enma Labs is the container around it.
